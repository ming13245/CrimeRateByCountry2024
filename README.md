# 빅데이터 분석 프로젝트:범죄

 ## 개요
 4대범죄, CCTV수, 경찰관수, 재범률, 한국-미국,영국,일본,프랑스,캐나다
 
 ## 목표 기능 및 작업 현황
  1) 가설설정
     - 범죄에 대한 가설 설정
       - 가설 종류 : CCTV수와 살인건수 비교 등..
    
  2) 산출물 작성
     - As-is map, To-be map 등등

 3) 데이터 전처리
    - 포맷을 통합
   
 ## 역할
 - 홍민기(조장) : 미국
 - 서유진      : 프랑스
 - 차다훈      : 영국
 - 김민주      : 일본
 - 주찬휘      : 캐나다
